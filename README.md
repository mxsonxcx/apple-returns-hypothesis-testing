# Apple (AAPL) Daily Returns – Statistical Hypothesis Testing

Statistical analysis of Apple Inc. (AAPL) daily stock returns in 2025, framed around realistic investment decisions.

## 🚪 What's inside?

Three hypothesis tests conducted on real market data:

- **z-test for mean (μ)** – does the average daily return exceed 0.05%?
- **Chi-square test for variance (σ)** – is daily volatility below 2.5%?
- **z-test for proportion (p)** – are positive-return days more frequent than 45%?

Each test is framed around a hypothetical investor scenario with
p-value interpretation and a plain-language decision.

## 📊 Data

Daily OHLCV quotes for AAPL, period 01.01.2025-31.12.2025,
sourced from stooq.com. Included in the "Data" sheet.
